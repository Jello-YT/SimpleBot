# SimpleBot
SimpleBot is a easy Discord.js bot which is designed for people starting to learn or people wanting the simple goods.

### Features

- Join and leave messages
- Uptime command
- Say (embed) commands
- Member count channel
- Auto-role adding on user join
- Timer command
- Moderation Commands
